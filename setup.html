Overview

The BLOPUP project integrates several tools and components that must be configured before use. This setup guide provides an overview of each requirement and links to detailed instructions.

Requirements

  1. Python Virtual Environment BLOPUP runs on a dedicated Python virtual environment that is required to ensure that all librearies are isolated and do not cause conflicts with other existing software. This virtual environment is needed to both users and developers. See Python Virtual Environment for setup steps.

  2. Sphinx Documentation System Sphinx is required to build the project documentation. If you develop on this project the Sphinx environment is needed. See Setting up the Sphinx Documentation Environment for setup steps.

  3. Deploying Sphinx Documentation to GitHub Enables automated generation and publishing of the documentation to GitHub Pages. This should have been set up on BLOPUP/winapp2 github repo, but still documented just in case. See Deploying Sphinx Documentation to GitHub for setup steps.
